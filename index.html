Configurações

Local

Por padrão este quadro funciona localmente, salvo apenas neste navegador. Para uso multiusuário real (todo mundo vendo o mesmo quadro e recebendo e-mails automáticos), publique o backend em Google Apps Script (arquivo Code.gs incluído no projeto) e cole a URL abaixo.

Ao entrar em uma coluna final, a tarefa continua visível no quadro até esse prazo. Você também pode arquivar manualmente pelo botão 📦 no card.
Usuários

Usuários são as contas de acesso ao sistema (e-mail e senha). Cada tarefa pode ainda ser delegada a um responsável diferente do usuário logado.

Nota sobre permissões: as restrições por cargo (admin/membro) organizam o uso da equipe dentro do aplicativo, mas não são uma barreira de segurança real — quem tiver a URL do backend consegue chamá-la diretamente. Para controle de acesso rigoroso seria necessário autenticação no Apps Script.
